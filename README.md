# word-processor

Форматирует академические тексты через ИИ: конвертирует LaTeX-формулы в нативные Word-формулы (OMML), применяет ГОСТ-стили, поддерживает настраиваемые скиллы для разных типов документов.

## Быстрый старт

```bash
# 1. Скопируй .env.example в .env и вставь свой ключ
cp .env.example .env

# 2. Запусти
docker compose up --build

# 3. Открой браузер
# http://localhost:5000
```

## Настройка (.env)

```env
# API ключ (обязательно)
API_KEY=your_key_here

# Модель — можно менять без пересборки
LLM_MODEL=qwen/qwen3.6-plus

# URL провайдера
LLM_BASE_URL=https://polza.ai/api/v1
```

Ключ получить: [polza.ai/dashboard](https://polza.ai/dashboard) или другого провайдера

## Как работает

```
Входной текст
    ↓
ИИ (Qwen) — структурирует текст, конвертирует формулы в LaTeX, чистит Unicode
    ↓ streaming (текст появляется сразу)
Markdown + LaTeX
    ↓
pandoc — Markdown → .docx (OMML формулы) или .pdf (xelatex)
    ↓
Готовый файл
```

## Скиллы

Скилл — это набор правил форматирования для конкретного типа документа. ИИ применяет их при обработке текста.

### Как создать скилл

Нажми **＋** рядом с "Скиллы" → заполни форму:

**Название** — коротко, например: `ГОСТ 7.32`, `ВКР ПГТУ`, `Тезисы`

**Описание** — одна строка о типе документа

**Инструкции** — конкретные правила форматирования. Примеры:

```
# ГОСТ 7.32
- Заголовки разделов: прописными, по центру, жирный
- Нумерация: 1, 1.1, 1.1.1
- ВВЕДЕНИЕ и ЗАКЛЮЧЕНИЕ — без номера, прописными
- Формулы нумеруются справа: (1), (2), ...
- Рисунок N — Название (ниже, по центру)
- Таблица N — Название (выше, слева)
- Литература по ГОСТ Р 7.0.5-2008
```

```
# Тезисы конференции
- Заголовок по центру, жирный, без нумерации
- Авторы и организация — курсивом, следующая строка
- Без оглавления
- Формулы нумеровать только если их больше двух
- Максимум 3 страницы
```

```
# IEEE статья
- Sections: Roman numerals I., II., III.
- Equations right-aligned: (1), (2)
- Figures: Fig. N. below, centered
- References: IEEE style [1], numbered list
```

Инструкции пиши на русском или английском — ИИ понимает оба языка.

## Запуск без Docker

```bash
pip install -r requirements.txt
python3 create_reference.py   # один раз
python3 app.py
```

Требования: Python 3.11+, pandoc, xelatex (для PDF).

## Тесты

```bash
pip install requests python-docx
python3 test_app.py
```
