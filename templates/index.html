            Как писать инструкции для скилла
          

          ▾
        

        

          
Что такое скилл? Набор правил, которые ИИ применит при форматировании. Пиши конкретно — чем точнее, тем лучше результат.

          
Что описывать:

          — Заголовки (прописные/строчные, центр/левый край, нумерация)

          — Нумерацию формул: сквозная (1),(2) или по главам (1.1),(1.2)

          — Подписи к рисункам (где, какой текст, курсив или нет)

          — Подписи к таблицам (выше/ниже, формат)

          — Список литературы (ГОСТ Р 7.0.5, IEEE, APA...)

          — Специфичные требования журнала / вуза


          
Пример — ГОСТ 7.32

          
- Заголовки разделов: прописными, по центру, жирный
- Нумерация: 1, 1.1, 1.1.1
- ВВЕДЕНИЕ и ЗАКЛЮЧЕНИЕ без номера, прописными
- Формулы нумеруются: (1), (2), ...
- Рисунок N — Название (ниже, по центру)
- Таблица N — Название (выше, слева)
- Литература по ГОСТ Р 7.0.5-2008


          
Пример — Тезисы конференции

          
- Заголовок по центру, жирный, без нумерации
- Авторы и организация — курсивом, следующая строка
- Без оглавления
- Формулы нумеровать только если их больше двух
- Максимум 3 страницы
- Не более 10 источников в литературе


          
Пример — IEEE

          
- Sections: Roman numerals I., II., III.
- Subsections: A., B., C. — italic inline heading
- Equations right-aligned: (1), (2)
- Figures: Fig. N. below, centered
- Tables: TABLE N above, small caps
- References: IEEE style [1], numbered list


          

            Инструкции уходят напрямую в системный промпт. Шрифт и поля задаются в настройках — не нужно описывать их здесь.
